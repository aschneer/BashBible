# Networking

```bash
# Get IP address and network info
ifconfig
# or (following are all the same)
ip address
ip addr
ip add
ip a
```

```bash
# See hostname and all IP addresses associated
# with the client.
hostname
hostname -I
```

```bash
# List all nodes on the network and their info.
nmap 10.133.0.*
	# Lists all nodes within this subnet.
```
